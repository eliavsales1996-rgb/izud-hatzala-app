# Ichud Hatzala Plus

A React Native app built with Tailwind CSS using NativeWind.

## Getting Started

1. Install dependencies: `npm install`
2. Start the Metro bundler: `npm start`
3. Run on Android: `npm run android`
4. Run on iOS: `npm run ios`

## Dashboard Screen

The app features a Dashboard screen with:
- Header with app title
- Emergency status card
- Active calls counter
- Emergency call and volunteer login buttons

## Technologies

- React Native
- TypeScript
- Tailwind CSS (via NativeWind)
